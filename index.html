Teste de Performance com Api PageSpeedInsights

Laboratório de performance do seu site com Api PageSpeedInsights

O Page Insights Speed é uma ferramenta do Google que permite verificar a eficiência do seu site, seguindo as recomendações do Google para garantir um bom desempenho tanto em SEO quanto para evitar que os usuários saiam do site devido à lentidão. Essa ferramenta avalia dispositivos tanto em desktop (computadores desktop, notebooks, etc.) quanto em dispositivos móveis (celulares, tablets, etc.).

Há alguns pontos importantes a serem observados, como as siglas O PSI informa as experiências de primeira exibição de conteúdo (FCP) dos usuários reais, latência na primeira entrada (FID, na sigla em inglês), entre outras. Embora eu não vá me aprofundar em explicações desses termos aqui, você pode consultar a Documentação do PageSpeedInsights para obter informações mais detalhadas e coletar as informações necessárias.

Esse projeto foi contruido com linguagem Python em que consiste em gerar um arquivo em CSV para utilizá-lo como base de dados em uma ferramenta de dataviz de sua preferência. Aqui, usaremos o PowerBI (em construção) como exemplo.
